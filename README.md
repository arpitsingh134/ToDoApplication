# Task Creating App

> **ToDO  is a java advance based application which use spring MVC and hibernet 5.**

## Features

- Anyone can create ,update ,delete and view tasks  with  simple UI 

---------

### Technology used in this project

#### Web Page design ####
- Bootstrap 4.0
- html



#### Backend technologies ####

- **`Spring MVC`**
- **`Java Core`**
- **`Servlets`** 
- **`JSP`**
- **`Hibernet 5`**
- **`Tomcat Server 9.0`**

#### Database ####

- **`MySql`**
